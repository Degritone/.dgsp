# .dgsp
A custom image format for sprites with 1-16 colors, including transparency. 20%-80% file sizes compared to PNG, even smaller than PNG-8 in most cases. Includes on-the-fly color changing. Not super optimized. This could definitely be done with WebGL, I was just too lazy to figure this out.
